온라인 투표

4차 산업혁명의 신기술로 불리는 불록체인이 금융거래와 물류, 의료, 행정 서비스에 이어 전자투표까지 그 적용범위를 넓혀가고 있다.
블록체인은 중앙 서버에 거래기록을 저장하지 않고 거래에 참여하는 개별 참여자의 서버들이 모여 네트워크를 유지 관리하는 분산형 원장기술을 말한다.
특히 블록체인방식은 개별 참여자간 정보공유와 상호검증으로 투표의 위변조 및 조작을 방지에 용이하며 익명성 또한 보장된다,

사용코드

pragma solidity ^0.4.19;

contract A{
    uint public num = 10;
    function getNum() constant external returns (uint){         return num;
    }
}
